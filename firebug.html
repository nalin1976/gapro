Disabling Firbug Extensions For Better System Performance

Firebug can cause performance issues and unusual system errors with ePlan ERP Web, so we recommend disabling it for ePlan ERP.

Windows or Linux Users

  1. Click the green or red icon in the bottom right corner of the browser window to open Firebug.
  2. Click the bug icon in the top left corner of Firebug and select Disable Firebug for ePlan .
If you'd like to keep Firebug running, you can improve ePlan ERP performance by following these steps:
  1. Click the green or red icon in the bottom right corner of the browser window to open Firebug.
  2. Click the Console tab.
  3. Select Options.
  4. Uncheck Show XMLHttpRequests.
  5. Click the Net tab.
  6. Select Options.
  7. Check Disable Network Monitoring.

Mac Users

  1. Click the green or red icon in the bottom right corner of the browser window to open Firebug.
  2. Click the bug icon in the top left corner of Firebug and select Disable Firebug for ePlan .

If disabling Firebug for ePlan ERP Web doesn't improve performance results, you may have to entirely disable Firebug.

If you need further assistance on this please do not hesitate to contact ePlan ERP development team.
